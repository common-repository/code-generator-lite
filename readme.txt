Create code generators.

== Description ==

This plugin allows you to create code generators for Xbox Live codes, Amazon codes, Minecraft codes, etc. In order for a user to obtain a code, they must go to a url, that you provide, to obtain a password. Once they obtain a password, they enter it in the generator to get a code.

**Lite Features**<br />
-Create 1 generator.<br /><br />
-Create 10 codes.<br />		
-Create 3 URLs per code.<br />	
-URLs are randomized.<br />	
-Customize generator's style.<br />	
-Generators can be added on posts or pages.<br /><br />
-Codes can only be accessed by one person at a time. A code becomes active once one of its URLs are accessed. If a password isn't entered, the code becomes non-active in an hour.<br />			
-Once a password is entered, the code and its URLs are deleted.

**Pro Features**<br />
-Create unlimited generators.<br />	
-Create unlimited codes.<br />	
-Create unlimited URLs.<br />	
-URLs are randomized.<br />	
-Each generator has its own style.<br />	
-Generators can be added on posts or pages.<br />	
-Codes can only be accessed by one person at a time. A code becomes active once one of its URLs are accessed. If a password isn't entered, the code becomes non-active in an hour.<br />	
-Once a password is entered, the code and its URLs are deleted.
